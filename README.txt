Vending Machine!

To run:
./vm <stock csv file> <coins csv file>

	- Files must be properly formatted.


		Main Menu:
		1) Purchase Product
		2) Display Products
		3) Save and Exit
		Supplier-Only Menu:
		4) Add Product
		5) Remove Product
		6) Display Coins
		7) Re-stock Products
		8) Re-stock Coins
		9) Abort

		Select your option (1-9): 


When purchasing a product, coins must be entered by value in cents.
e.g. $1 coin = 100

To abort any function, press enter when prompted for input.
